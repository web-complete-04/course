Recap ex 03

The program will generate 2 random numbers and will ask the user to provide the value of their sum

This will repeat until the user has 3 wrong answers

For each answer, the program will validate the user's answer (Correct or incorrect)

If the user's answer is incorrect, the program will display the correct value of the sum

If the user's answer is correct, the program will increase the score of the user

Before ending the program, it will display the user's score
